# Installation

## Using pip

The recommended way to install LlamaX is via pip:

```bash
pip install llamax
```

## From Source

You can also install LlamaX from source:

```bash
git clone https://github.com/llamasearchai/llamax.git
cd llamax
pip install -e .
```

## Dependencies

LlamaX requires:

- Python 3.8+
- Other dependencies are automatically installed by pip
